rdis

Limited Closed Release Alpha

We are looking for people to test rdis. Send your distro/architecture to (endeavor @ rainbowsandpwnies dot com) and we will send you a binary with instructions on how to run it. We ask for your thoughts, requests, criticisms and bug reports so we can make rdis better.

Rdis, the Rainbowsandpwnies DISassembler, is a binary analysis platform under development by rainbowsandpwnies.

What rdis will do:

  • Analyze multiple binary formats with multiple instruction sets.
  • Translate assembly into an IR for analysis.
  • Create interactive Control Flow Graphs.
  • Provide a scriptable interface through Lua.

What works now:

  • ELF and PE binaries.
  • x86 and x86-64 instruction sets.
  • Identification of functions through multiple methods, including recursive disassembly.
  • Control Flow Graphs
  • User defined entry points for recursive function discovery and disassembly
  • Interactive control flow graphs.
  • Lua scripting.

What's planned for the first release:

  • ELF, PE and Mach-O binaries.
  • x86, x86-64 and ARM assembly.
  • Translation of assembly to IR.
  • Remote debugging.
  • Realtime collaborative analysis.

Other Significant Details:

  • rdis requires Gtk3 which is not currently supported by backtrack. Get a real distro.
  • We do not know under what terms rdis will be released.
  • Release date is unknown. Expect early 2013.
